Statistics

Att: Attempts

Number of attempts to ring in, as given in the box score.

BuzC: Correct on Buzz

Correct answers on buzz. Equivalent to COR from the box score, minus any correct Daily Doubles.

Buz$: Score on Buzz

Score using only buzz-in responses. Equivalent to actual score and to Coryat score, except it ignores Daily Doubles entirely.

CCon: Contestant Contention

An estimate of the probability that any individual correct response this contestant gave blocked an opponent from buzzing in.

DD: Daily Doubles Found

Daily Doubles found by the player.

DD+: Daily Doubles Over Expectation

An estimate of how many more Daily Doubles a player found versus an expectation that each correct response in a round is equally likely to lead to finding a Daily Double. This statistic is zero-sum: all positive DD+ must be balanced by negative DD+ for another contestant.

GCon: Game Contention

An estimate of the probability that any individual correct response (for any contestant) block an opponent from buzzing in.

Solo: Buzzes that are uncontested

An estimate of how many correct responses did not block an opponent's attempt at a correct response. That is, it is an estimate of how many clues the contestant alone was correct on.

Speed: Buzzes created through speed

An estimate of how many additional response attempts a player gains through speed, or gains through opponents' lack of speed. This statistic is zero-sum: all positive Speed must be balanced by negative Speed for another contestant.
